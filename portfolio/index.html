Website Design and Development Basics

HTML

This is an example of HTML code. HTML stands for hyper-text markup language. HTML is what gives a website its content and allows for the structure of the content. HTML is used in any website that you encounter on the web. The basic syntax used is to open, and to close whatever element you are working on (header, footer, p, li, etc.)

CSS

This is an example of CSS code. CSS stands for cascading style sheets. CSS allows us to style a website by giving it color and size. We are able to determine where each element will be placed on the website. Without CSS our websites would look very strange!
